# Changelog

All notable changes to MailForge will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-14

### Added

- Initial release
- Go backend API with comprehensive email diagnostics
- React 19 WebUI with Radix UI components
- Email authentication: SPF, DKIM, DMARC validation
- DNS analysis: lookups, DNSSEC, blacklist checking
- Network diagnostics: port scanning, traceroute, HTTP headers
- TLS security analysis for SMTP connections
- WebSocket-based real-time monitoring
- Email deliverability scoring
- Comprehensive test suite (70+ tests)
- GitHub Actions CI/CD pipelines
- Pre-commit hooks and Makefile for development
- Docker support for containerized deployment

### Features

- **API Endpoints**: 15+ diagnostic endpoints
- **WebSocket Monitor**: Real-time diagnostic streaming
- **Tool Suite**: 10+ specialized email/network tools
- **Test Coverage**: Unit and integration tests for all tools
